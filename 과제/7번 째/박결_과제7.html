HTML 태그 소개

속성으로 분류

  1. block 속성

    block 속성은 하나의 태그가 한 줄의 영역을 차지한다.

    • <p>태그는 문단을 정의합니다.
    • <div >태그는 문서의 섹션을 만들거나 영역을 나눌 때 사용합니다.

  2. inline 속성

    inline 속성은 하나의 태그가 자신의 영역외의 영역은 차지하지 않는다.
    따라서 margin값이나 padding값 등 영역을 부여하는 css 속성을 줄 수 없다.

    • <span >태그는 인라인 요소 그룹화를 정의합니다.
    • < a>태그는 하이퍼 링크를 정의합니다.
